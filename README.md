# basic-webpack-boilerplate
A basic config of webpack bundler

Don't forget the: `npm install`

then use it :)

**webpack Plugins that used for this boilerplate:**
```
-css-loader
-file-loader
-html-loader
-html-webpack-plugin
-mini-css-extract-plugin
-node-sass
-sass-loader
-style-loader
-webpack-dev-server
```
